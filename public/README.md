# Trinity Pharma — Public Assets

Drop static assets into the folders below. They are tracked with `.gitkeep` so
the structure persists even while empty.

## Folder map

- `video/hero/` — homepage hero background video.
  - **`hero.mp4`** + **`hero.webm`** — the hero video (provide both for browser coverage).
  - **`hero-poster.jpg`** — first-frame poster. **The preloader preloads this exact file**
    (`/video/hero/hero-poster.jpg`) as part of its completion gate.
- `video/story/cannabis/` — scroll-story video clips for the `/cannabis` page.
- `video/story/beschaffung/` — scroll-story video clips for the `/beschaffung` page.
- `frames/cannabis/` — image-sequence frames (if the cannabis story uses frame scrubbing).
- `logos/` — partner / press logos.
- `brand/` — brand marks, OG images, favicons, and other brand assets.
